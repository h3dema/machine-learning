## Instalação de dependencias


O código neste repositório foi testado com Puthon versão 3.


1. No linux

```
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt
```

2. No windows

```
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```
